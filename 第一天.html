Web

World Wide Web,全球局域网，也叫万维网

基于超文本和HTTP

分布式图形信息系统

HTML

超文本：文本+非文本（图片，视频，音频等）

标记：也叫标签，分单标签和双标签。

语言：编程语言

CSS

Cascading Style Sheets,层叠样式表

用来表现HTML或XML

静态修饰网页或配合脚本语言(JavaScript等)对网页元素进行控制

JavaScript

基于原型编程、多范式的动态脚本语言，函数编程风格

由ECMAScript（语法和对象），DOM（文档对象模型），BOM（浏览器对象模型）组成

嵌入动态文本于HTML页面。对浏览器事件做出响应。读写HTML元素。在数据被提交到服务器之前验证数据等
